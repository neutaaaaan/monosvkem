Monosvkem, a color scheme with a unique touch.

==============================================================================
Color scheme options					Monosvkem-options*

						*g:Monosvkem_transp_bg*
Set to 1 if you want a transparent background. Takes effect only in the
terminal.
>
	let g:Monosvkem_transp_bg = 0
<
